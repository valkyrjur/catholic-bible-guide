# beginner-html-site-styled
A simple one page website created to help complete beginners learn HTML basics, which in this repo has also been styled to help beginners learn CSS basics. The styling is explained over the course of [https://developer.mozilla.org/en-US/Learn/Getting_started_with_the_web/CSS_basics](https://developer.mozilla.org/en-US/Learn/Getting_started_with_the_web/CSS_basics).

[Run the example live](http://mdn.github.io/beginner-html-site-styled/).

## 📦 How to Run Locally

Follow these simple steps to view and edit this site on your own computer:

### 1. 💻 Clone the Repository
Open your terminal or Git Bash and type:

```bash
git clone https://github.com/mdn/beginner-html-site-styled.git
```
2. 📂 Go into the Project Folder

```bash
cd beginner-html-site-styled
```

3. 📜 Open the Website

Locate the file called index.html inside the project folder and open it in your web browser.

You can do this by:

double‑clicking on the file, OR

right‑click → Open with → choose your browser (Chrome, Firefox, etc.)

4. 🔄 Make Edits

You can edit the HTML and CSS files using any text editor:

📝 VS Code

🧑‍💻 Atom

📄 Notepad / TextEdit

Just save your changes and refresh your browser to see them.

💡 Tip for beginners

If you want to preview the site using a local server (recommended for learning):

-Install Visual Studio Code

-Add the Live Server extension

-Open the project in VS Code

-Click “Go Live” at the bottom right

This will automatically open the site in your browser and refresh it as you edit the code. 😊
